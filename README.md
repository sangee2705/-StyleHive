# 🛍️ StyleHive - Fashion E-commerce Website

StyleHive is a modern, responsive fashion e-commerce website built using **HTML**, **CSS**, and **JavaScript**. It provides a clean shopping experience with responsive layouts, product search, category filtering, and an attractive user interface.

---

## 📌 Project Overview

StyleHive is designed as a fashion shopping website where users can browse products, search for items, filter by category, and contact the store.

---

## 🚀 Features

- 🏠 Responsive Home Page
- 🛍️ Collections Page
- 📞 Contact Us Page
- 🔍 Product Search
- 🏷️ Category Filter using JavaScript
- 📱 Mobile Responsive Design
- 🎨 Modern UI Design
- ✨ Hover Animations
- 📦 Product Cards
- 📧 Newsletter Section
- 🌐 Social Media Links
- 📌 Sticky Navigation Bar

---

## 💻 Technologies Used

- CSS3
- JavaScript
